Personal Webpage


This site is my personal webpage and is designed to act as a digital CV for future employers and anyone who's interested to learn more about me.

This project started off as an introduction to the world of programming. Starting off with primarily frontend languages, such as HTML and css, the goal was to create something visual I could test my new learnt skills, while also creating a digital CV for myself. Having done my research, I felt this was a perfect place to start my coding journey, while also allowing me to constantly revisit and hone my skills as my career progresses.

Having started with HTML and css, i've since learnt and introduced Javascript in order to make my site more mobile friendly and also show off my basic skill in backend development. The main buik of my backend skill set will be showcased in other projects, all of which will be listed in the projects section of my website.
